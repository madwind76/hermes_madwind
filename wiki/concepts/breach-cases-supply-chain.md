## 🔗 공급망 공격 (4건)

### 17. Kaseya VSA (2021) ⭐⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | CVE-2021-34746 — 인증 우회 → REvil 랜섬웨어 대규모 배포 |
| **실습 도구** | 커뮤니티 Kaseya VSA 취약 컨테이너 |
| **실습 내용** | MSP 환경 모니터링, 패치 검증, 릴리스 검증 절차 |
| **교육 포인트** | MSP 공급망 위험, 원격 관리 도구의 이중성 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2021-07-02 |
| **발견일** | 2021-07-02 (Kaseya 보안팀) |
| **공격 타임라인** | REvil이 Kaseya VSA 0-day로 자격증명 우회 → 악성 업데이트를 정상 업데이트 채널로 푸시 → **60개 MSP → 1,500개 기업 2시간 만에 감염** → 스웨덴 Coop (800개 매장 폐쇄) → 2021-07-04: Kaseya 긴급 패치 |
| **피해 규모** | 60개 MSP, 1,500개 기업. 스웨덴 Coop 식료품점 800개 폐쇄. REvil 7,000만 달러 복호화 키 제안 |
| **발견 경로** | Kaseya 보안팀이 고객 지원 티켓 해킹 신고 급증 → 이상 활동 감지 → VSA 인스턴스 전면 격리 |
| **배후** | REvil (Pinchy Spider). FBI/CISA 공식 귀속 |

### 18. 3CX DesktopApp (2023) ⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | Electron 앱 업데이트 변조 → 공급망 손상 |
| **실습 도구** | CI/CD 파이프라인 시뮬레이션 + 악성 DLL 주입 |
| **실습 내용** | 파일 무결성 모니터링, 디지털 서명 검증, VSA 분석 |
| **교육 포인트** | Electron/데스크탑 앱 공급망, 코드 서명의 신뢰 문제 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2023-03-22 |
| **발견일** | 2023-03-22 (CrowdStrike, Mandiant) |
| **공격 타임라인** | 3CX 빌드 시스템 침투 (공급망) → Electron 앱 업데이트에 악성 DLL(ffmpeg.dll 위장) 포함 → 60만 고객에 정식 배포 → CrowdStrike Falcon Sensor가 비정상 네트워크 연결 감지 |
| **피해 규모** | 약 60만 고객 잠재 피해. 많은 기업 감염. 최소 6개 하위 공급망 파급 (2차 피해) |
| **발견 경로** | CrowdStrike Falcon Sensor가 3CX 앱의 비정상 네트워크 연결 감지 → 분석 → 맬웨어 식별 |
| **배후** | Lazarus Group (북한) — FBI 귀속, Kaspersky 분석 |

### 19. CodeCov Bash Uploader (2021) ⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | CI/CD 스크립트 변조 → 자격증명 탈취 |
| **실습 도구** | GitLab CI + 스크립트 인젝션 Docker |
| **실습 내용** | CI/CD 보안 강화, 시크릿 스캐닝, 스크립트 변경 감사 |
| **교육 포인트** | CI/CD 파이프라인이 새로운 공격 표면 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2021-01-31 (초기 침투) |
| **발견일** | 2021-04-15 (내부팀) |
| **공격 타임라인** | 공격자가 Codecov Docker 이미지 빌드 과정 침투 → Bash Uploader 스크립트 수정(환경변수 유출 코드 삽입) → 모든 CI/CD 실행 시 고객의 AWS/GitHub 키 유출 → **2개월간 무증상 지속** |
| **피해 규모** | 수천 개 조직 (Twilio, GoDaddy, Rapid7, HashiCorp, Confluent 등). 유출된 키로 추가 2차 공격 가능 |
| **발견 경로** | Codecov 보안팀이 Bash Uploader 스크립트의 예기치 않은 출력 감지 → 포렌식에서 감염 확인 |
| **배후** | 아직 **공식 귀속 없음**. 2022년 FBI/CISA 경보: \"악의적 사이버 행위자\" |
