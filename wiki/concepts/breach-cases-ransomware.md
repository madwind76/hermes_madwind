# 실제 침해 사례 — 랜섬웨어 (주요 사례)

> 실습 환경(Docker/VM)으로 재현 가능한 실제 사이버 공격 사례 종합.
> 본 페이지는 [[real-world-breach-cases]]의 하위 페이지입니다.
> 평가 기준: ⭐⭐⭐(즉시 사용) / ⭐⭐(일부 구성) / ⭐(PoC 수준)

## 🟠 Ransomware (6건)

### 1. WannaCry (2017) ⭐⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | EternalBlue (MS17-010) — SMBv1 RCE, 웜 확산 |
| **CVE** | CVE-2017-0144 |
| **실습 도구** | Vulhub `smb/ms17-010`, Metasploit `exploit/windows/smb/ms17_010_eternalblue` |
| **실습 내용** | SMB 취약점 스캔 → 익스플로잇 → 랜섬웨어 암호화 관찰 → 네트워크 분할 탐지 |
| **Docker** | Vulhub 공식 지원, Metasploitable2 포함 |
| **교육 포인트** | 패치 관리의 중요성, SMB 프로토콜 격리, 웜 확산 메커니즘 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2017-05-12 |
| **발견일** | 2017-05-12 (당일) |
| **공격 타임라인** | 2017-04-14: Shadow Brokers가 NSA 도구(EternalBlue) 유출 → 2017-05-12: 전 세계 급속 확산 (SMBv1 웜) → 2017-05-13: Marcus Hutchins가 kill switch 도메인 등록으로 확산 저지 |
| **피해 규모** | 150개국, 20만+ 시스템. 영국 NHS(병원 80곳 마비, 수술·진료 중단), Telefónica, FedEx, Deutsche Bahn, Renault(공장 중단). 총 피해 약 40억 달러 (보험·생산성 손실·복구 비용) |
| **발견 경로** | 보안 연구원 Marcus Hutchins가 웜 샘플 분석 중 kill switch 도메인(등록되지 않은 `.tu` 도메인) 발견 → 등록하여 확산 저지 |
| **배후** | Lazarus Group (북한) — FBI, NSA, 영국 NCSC 공식 귀속 |

### 2. NotPetya (2017) ⭐⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | EternalBlue + Mimikatz + WMIC, 위장 랜섬웨어 (실제는 와이퍼) |
| **실습 도구** | WannaCry 실습 환경 + Caldera 에뮬레이션 |
| **실습 내용** | MFT/MBR 변조 분석, 볼륨 섀도 복사본 삭제 관찰, 포렌식 |
| **교육 포인트** | 랜섬웨어와 와이퍼의 차이, 백업 전략, 공급망을 통한 전파 (M.E.Doc) |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2017-06-27 |
| **발견일** | 2017-06-27 (당일) |
| **공격 타임라인** | 2016~2017: 우크라이나 회계 SW M.E.Doc 업데이트 서버에 백도어 삽입 → 2017-06-27: NotPetya 변종 배포 → 우크라이나 정부·기업 마비 → 전 세계로 확산 (EternalBlue + WMIC + SMB) |
| **피해 규모** | Maersk(76억 달러 손실, 10일간 컨테이너선 마비), Merck(13억 달러, 백신 생산 중단), Mondelez(10억 달러 소송), FedEx TNT(3억 달러), 체르노빌 방사선 모니터링 정지. **총 피해 100억 달러 이상** (사상 최대) |
| **발견 경로** | 우크라이나 사이버 경보팀 + ESET 연구진이 M.E.Doc 업데이트 서버를 통한 감염 경로 확인 |
| **배후** | Russian GRU Sandworm (APT44) — 미국·영국·EU 공식 귀속. 우크라이나 전쟁 전 사이버 무기 테스트 성격 |

### 3. REvil / Sodinokibi (2020-2021) ⭐⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | Pulse Secure VPN 취약점 (CVE-2019-11510) + RaaS 모델, 이중 협박 |
| **CVE** | CVE-2019-11510 |
| **실습 도구** | Pulse VPN Docker (GitHub), Caldera + Atomic Red Team |
| **실습 내용** | VPN 게이트웨이 공격, 데이터 암호화 + 탈취 정찰, C2 통신 탐지 |
| **교육 포인트** | VPN 취약점 패치, 이중 협박 대응, RaaS 비즈니스 모델 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2021-05-30 (JBS) / 2021-07-02 (Kaseya) |
| **발견일** | 2021-05-30 (JBS) / 2021-07-02 (Kaseya 보안팀) |
| **공격 타임라인** | 2019년 말 활동 시작 → 2021-05-30: JBS USA 공격 (세계 최대 육류 가공업체, 소·돼지 가공 공장 전면 마비) → 2021-07-02: Kaseya VSA 공급망 공격 (60개 MSP → 1,500개 기업 2시간 만에 감염) → 2021-10: 러시아 압박으로 REvil 사이트 폐쇄 |
| **피해 규모** | JBS: 1,100만 달러 비트코인 지불 (미국 쇠고기 생산 1일 재개). Kaseya: 800~1,500개 비즈니스. Apple(Quanta) 협박 5,000만 달러. 15만+ 시스템 감염. |
| **발견 경로** | JBS: 내부 IT 시스템 이상 감지 → 사업 중단 결정. Kaseya: VSA 업데이트 무결성 이상 탐지 → FireEye/Mandiant 합동 분석 |
| **배후** | REvil Group (Pinchy Spider, UNKN) — 러시아어권. 매월 6,000만 달러 수익 추정. 2022년 러시아 FSB 해체 발표 |

### 4. LockBit (2020-현재) ⭐⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | RaaS, Log4Shell (CVE-2021-44228), Exchange CVEs 활용 |
| **실습 도구** | Vulhub `log4j/CVE-2021-44228` + LockBit 흉내 Atomic Tests |
| **실습 내용** | Log4j 스캔 → 익스플로잇 → LockBit 유사 페이로드 배포 → 데이터 유출 감지 |
| **교육 포인트** | 멀티 벡터 공격 체인, WAF/EDR 로그 기반 탐지, 협박 노트 분석 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2020-09 (LockBit 1.0) ~ 현재 |
| **발견일** | 지속적 발견 (보안 연구자 + 법집행) |
| **공격 타임라인** | 2020-09: LockBit 1.0 등장 → 2021-06: 2.0 (BlackMatter 규모) → 2022-06: 3.0 (자동 분산 협상) → 2023-01: Royal Mail (프린터로 협박 편지 출력) → 2023-11: ICBC 금융 공격 (美 국채 청산 차질) → 2024-02: Operation Cronos (전 세계 법집행이 인프라 장악, 운영자 실명 공개) |
| **피해 규모** | 2,000+ 피해자. CDW, Royal Mail, Boeing, ICBC 법인. 총 1.2억 달러 이상 갈취 추정 (Trustwave) |
| **발견 경로** | 법집행 작전 (영국 NCA 주도) + 보안 연구자 그룹(VX-Underground, Trend Micro) 모니터링 |
| **배후** | LockBitSupp (Dmitry Khoroshev? 라트비아 출신?). 수사 진행 중 |

### 5. Conti (2020-2022) ⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | BazarLoader + Cobalt Strike + 유출된 내부 플레이북 |
| **실습 도구** | Atomic Red Team (Conti TTPs), Caldera 사용자 지정 프로파일 |
| **실습 내용** | 피싱 → C2 → 랜섬웨어 전개 시뮬레이션, SOC 알람 튜닝 |
| **교육 포인트** | 내부 문서 유출의 위험, RaaS 운영 방식, Cobalt Strike 탐지 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2021-05-14 (아일랜드 HSE) / 2022-02 (내부 유출) |
| **발견일** | 2021-05-14 (HSE) / 2022-02 (내부 채팅 유출) |
| **공격 타임라인** | 2020년부터 활동 (가장 큰 RaaS) → 2021-05-14: 아일랜드 HSE 공격 (병원 시스템 80% 마비, 방사선 치료·암 수술 연기) → 2022-02: 우크라이나 침공 직후 Conti가 러시아 지지 선언 → 반발로 우크라이나 해커가 내부 대화방 채팅 6만 건 유출 → 전체 TTPs 공개 |
| **피해 규모** | 아일랜드 HSE: 200만 환자 기록, 6,000만 유로 피해. 200+ 조직. 총 갈취 1.8억 달러 추정 |
| **발견 경로** | HSE: IT 이상 탐지 → 랜섬노트 발견. 내부 유출: 우크라이나 연구원이 채팅 분석 → Tactics/Playbook/소스코드 전면 공개 (보안 업계에 큰 기여) |
| **배후** | Conti Team (Wizard Spider, 러시아어권). 일부 멤버 Black Basta/LockBit 등으로 이적 |

### 6. DarkSide / Colonial Pipeline (2021) ⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | VPN 침투 → OT-IT 연결망 분리 부재 → 랜섬웨어 |
| **실습 도구** | GRFICS (ICS 시뮬레이션) + 랜섬웨어 스크립트 |
| **실습 내용** | IT-OT 네트워크 분할 평가, 백업 복구 절차, 위기 커뮤니케이션 |
| **교육 포인트** | OT 보안의 중요성, 연료 공급망 마비의 사회적 영향, 분할 설계 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2021-05-06 |
| **발견일** | 2021-05-06 (당일) |
| **공격 타임라인** | Colonial Pipeline 지불 시스템 침투 → 계정 탈취 → VPN 접속 → 랜섬웨어 배포 → 사측 즉시 5,500마일 파이프라인 폐쇄 → 2021-05-08: FBI 연락 → 2021-05-10: 75 BTC (440만 달러) 지불 → 2021-05-12: 파이프라인 재가동 → 2021-05-15: FBI가 일부 비트코인 압수 |
| **피해 규모** | 5,500마일 파이프라인 5일 정지. 동부 해안 연료 대란 (주유소 품절, 비행기 운항 차질, 비상사태 선포). Colonial 440만 달러 지불, 총 피해 약 980만 달러 |
| **발견 경로** | Colonial Pipeline IT팀이 지불 시스템 이상 감지 → 랜섬웨어 노트 발견 → 즉시 파이프라인 전면 차단 |
| **배후** | DarkSide (러시아어권 RaaS). 이후 REvil로 재편. FBI가 운영자 일부 체포 |
