# 실제 침해 사례 — 클라우드 보안 (주요 사례)

> 실습 환경(Docker/VM)으로 재현 가능한 실제 사이버 공격 사례 종합.

> 평가 기준: ⭐⭐⭐(즉시 사용) / ⭐⭐(일부 구성) / ⭐(PoC 수준)

## ☁️ 클라우드 보안 (3건)

### 20. Tesla K8s (2018) ⭐⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | 인증되지 않은 Kubernetes Dashboard → 암호화폐 채굴기 배포 |
| **실습 도구** | Minikube + 취약 K8s Dashboard 설정 |
| **실습 내용** | K8s RBAC 감사, 네임스페이스 격리, 컨테이너 이미지 스캔 |
| **교육 포인트** | K8s 기본 보안 설정의 중요성, 암호화폐 채굴 탐지 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2018-02 |
| **발견일** | 2018-02 (RedLock) |
| **공격 타임라인** | 공격자가 인증 없는 Tesla Kubernetes Dashboard 발견 → 클러스터 admin 접근 → AWS 크리덴셜 노출된 S3 버킷 발견 → 암호화폐 채굴기 배포 |
| **피해 규모** | Tesla 클라우드 인프라 일부 노출. 암호화폐 채굴 비용 Tesla 부담. 데이터 유출 가능성은 낮음 |
| **발견 경로** | RedLock(팔로알토 네트워크) 연구팀이 클라우드 감사 중 K8s Dashboard 공개 + 채굴 활동 발견 |
| **배후** | 미확인 해커 (암호화폐 채굴 목적, 국가 배후 아님) |

### 21. Microsoft Midnight Blizzard (2023) ⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | 패스워드 스프레이 → 레거시 테스트 테넌트 침투 |
| **실습 도구** | Azure AD 시뮬레이션, Raikia 도구 |
| **실습 내용** | 피싱 시뮬레이션, MFA 강제, 레거시 계정 모니터링 |
| **교육 포인트** | 테스트 환경도 보호 대상, 패스워드 스프레이 탐지 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2023-11 (초기 접근) |
| **발견일** | 2024-01-11 (Microsoft 보안팀) |
| **공격 타임라인** | APT29가 MS 기업 계정에 패스워드 스프레이 성공 → 임직원 계정 탈취 → 소스 저장소·내부 도구·**사이버보안팀 이메일** 접근 (고객 보안 정보 포함 가능성) → 2024-01-19: SEC 규정으로 공개 |
| **피해 규모** | MS 소스코드 저장소, 내부 이메일(특히 사이버보안팀) 접근. 고객 데이터 직접 유출은 부인했으나 보안 정보 노출 가능성 |
| **발견 경로** | Microsoft 보안팀이 비정상 로그인 시도 + 사서함 접근 패턴 발견 → 포렌식 |
| **배후** | Midnight Blizzard (Nobelium, APT29) — 러시아 SVR |

### 22. Twilio (2022) ⭐⭐
| 항목 | 내용 |
|------|------|
| **공격 기법** | 소셜 엔지니어링 + SMS 피싱 → SMS API 키 탈취 |
| **실습 도구** | 피싱 시뮬레이션 + API 사용량 분석 |
| **실습 내용** | SMS 기반 MFA의 취약점, API 키 로테이션, 이상 API 호출 탐지 |
| **교육 포인트** | SMS MFA는 피싱에 취약, 다중 인증 요소 다양화 |

#### 사고 상세
| 세부 항목 | 내용 |
|----------|------|
| **발생일** | 2022-08-04 |
| **발견일** | 2022-08-04 (보안팀) |
| **공격 타임라인** | LAPSUS$가 Twilio 직원에게 SMS 피싱 전송 (\"Twilio 인증 만료\" 사칭) → 직원이 가짜 로그인 페이지에 자격증명 입력 + MFA 코드까지 전송 → 공격자가 내부 시스템 접근 → 100+ 고객 데이터 유출 → 2022-08-08: Twilio 공개 |
| **피해 규모** | 100+ 고객 (Authy, Signal 등). Signal: 1,900명 사용자 전화번호 유출 |
| **발견 경로** | Twilio 보안팀이 다수 직원의 피싱 신고 + 이상 로그인 동시 감지 |
| **배후** | LAPSUS$. 2023년 영국 경찰 체포, 기소 |
