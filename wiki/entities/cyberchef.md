## 정의
다양한 인코딩/디코딩/변환 작업을 빠르게 수행할 수 있는 웹 기반 데이터 분석 도구입니다.

## Web CTF에서의 역할
- Base64, URL encoding, hex를 빠르게 판별합니다.
- 웹 응답이나 HTML attribute 값을 디코딩할 때 유용합니다.
- 반복적으로 문자열 포맷을 시험할 수 있습니다.
